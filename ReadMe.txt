Для того, чтобы запустить проект необходимо:
1. В папке DataBaseScripts запустить файл CreateFullDataBase.sql для установки базы данных. Пhb желаниb запустить файл InsertSomeData.sql после этого, чтобы наполнить базу даннными.
2. В папке CookBook/CookBook.Web в файле Web.config изменить имя сервера в строке подключения
3. Публиковать сайт желательно так: localhost:4002 - для рекламного сервиса, localhost:4001 - для самого сайта
